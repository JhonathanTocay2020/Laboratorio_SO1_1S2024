Crear un 
cargo new app-cliente
cargo run

pagina: rocket.rs

Agregar en Dependencias  
rocket = { version = "0.5.0-rc.1", features = ["secrets", "tls", "json"] }
serde = { version = "1.0", features = ["derive"] }

Primero, intenta instalar pkg-config en tu sistema. Puedes hacerlo ejecutando el siguiente comando en tu terminal:
sudo apt install pkg-config

OpenSSL en tu sistema. Para Ubuntu/Debian, puedes instalar libssl-dev:
sudo apt install libssl-dev
